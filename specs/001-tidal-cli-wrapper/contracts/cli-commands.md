# CLI Commands Contract: Tidal CLI

**Date**: 2026-03-15
**Feature**: 001-tidal-cli-wrapper

## Global Options

| Option   | Type | Default | Description                              |
|----------|------|---------|------------------------------------------|
| `--json` | bool | false   | Output as structured JSON instead of text |

The `--json` flag must be placed **before** the subcommand group:
```
python tidal_cli.py --json search artist "Daft Punk"
```

## Commands

### `auth`

**Description**: Interactive OAuth authentication (human only).
**Usage**: `python tidal_cli.py auth`
**Arguments**: None
**Options**: None

**Stdout (success)**:
```
Open this link in your browser to log in:
https://login.tidal.com/authorize?...

Waiting for validation...
Authentication successful!
```

**Stdout (success, --json)**:
```json
{"status": "success", "message": "Authentication successful"}
```

**Stderr (error)**:
```
Error: Authentication failed or expired. Please try again.
```

**Exit code**: 0 (success) | 1 (error)

---

### `search artist <query>`

**Description**: Search for an artist by name in the Tidal catalog.
**Usage**: `python tidal_cli.py search artist "Daft Punk"`

| Parameter | Type   | Position    | Required | Description   |
|-----------|--------|-------------|----------|---------------|
| query     | string | argument #1 | yes      | Artist name   |

**Stdout (text)**:
```
ID: 7804  Name: Daft Punk
ID: 12345 Name: Daft Punk (Tribute)
```

**Stdout (JSON)**:
```json
[
  {"id": 7804, "name": "Daft Punk"},
  {"id": 12345, "name": "Daft Punk (Tribute)"}
]
```

**Stdout (no results)**: Empty line (text) or `[]` (JSON)
**Exit code**: 0 (success, even with no results) | 1 (error)

## `library` commands (library / favorites)

### `library add`

**Description**: Add an artist, album or track to the user's library.
**Usage**:
```
python tidal_cli.py library add --artist-id 7804
python tidal_cli.py library add --album-id 1234
python tidal_cli.py library add --track-id 56789
```

| Parameter   | Type | Position | Required             | Description          |
|-------------|------|----------|----------------------|----------------------|
| --artist-id | int  | option   | mutually exclusive   | Artist ID to add     |
| --album-id  | int  | option   | mutually exclusive   | Album ID to add      |
| --track-id  | int  | option   | mutually exclusive   | Track ID to add      |

Exactly one of the three parameters must be provided per call.

**Stdout (text)**:
```
Added artist "Daft Punk" to library.
Added album "Discovery" to library.
Added track "Around the World" to library.
```

**Stdout (JSON)**:
```json
{"status": "success", "type": "artist", "id": 7804, "name": "Daft Punk"}
{"status": "success", "type": "album", "id": 1234, "name": "Discovery"}
{"status": "success", "type": "track", "id": 56789, "name": "Around the World"}
```

**Stderr (error)**:
```
Error: You must provide exactly one of --artist-id, --album-id or --track-id.
Error: Artist not found (ID: 99999).
```

**Exit code**: 0 (success) | 1 (error)

---

## Common errors (all commands except `auth`)

| Condition                    | Stderr message                                                                                    | Code |
|------------------------------|---------------------------------------------------------------------------------------------------|------|
| Missing/invalid session      | `Error: Not authenticated. The system administrator must first run 'python tidal_cli.py auth' manually.` | 1    |
| Network error                | `Error: Unable to connect to Tidal. Check your network connection.`                               | 1    |
| Missing argument             | Auto-generated usage message by Typer                                                             | 2    |
