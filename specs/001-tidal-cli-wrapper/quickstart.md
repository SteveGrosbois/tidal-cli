# Quickstart: Tidal CLI

**Date**: 2026-03-15
**Feature**: 001-tidal-cli-wrapper

## Prerequisites

- Python 3.10+
- An active Tidal account with a valid subscription

## Installation

```bash
pip install tidalapi typer
```

## Authentication (one time only, by human)

```bash
python tidal_cli.py auth
# -> Displays a URL: open it in your browser
# -> Authorize the application
# -> The .tidal_session.json file is created automatically
```

## Usage by LLM Agent

### Search for an artist

```bash
python tidal_cli.py search artist "Daft Punk"
# ID: 7804  Name: Daft Punk

python tidal_cli.py --json search artist "Daft Punk"
# [{"id": 7804, "name": "Daft Punk"}, ...]
```

### Search for an album

```bash
python tidal_cli.py search album "Discovery"
# ID: 1234  Artist: Daft Punk  Year: 2001  Name: Discovery

python tidal_cli.py --json search album "Discovery"
# [{"id": 1234, "name": "Discovery", "artist": "Daft Punk", "year": 2001}, ...]
```

### Search for a track

```bash
python tidal_cli.py search track "Around the World"
# ID: 56789  Artist: Daft Punk  Name: Around the World

python tidal_cli.py --json search track "Around the World"
# [{"id": 56789, "name": "Around the World", "artist": "Daft Punk"}, ...]
```

### List playlists

```bash
python tidal_cli.py playlist list
# ID: abc-123  Name: AI Mix (3 tracks)

python tidal_cli.py --json playlist list
# [{"id": "abc-123", "name": "AI Mix", "num_tracks": 3}, ...]
```

### Create a playlist

```bash
python tidal_cli.py playlist create --name "AI Mix" --desc "Generated by Claude"
# Created playlist: ID: ghi-789

python tidal_cli.py --json playlist create --name "AI Mix" --desc "Generated by Claude"
# {"id": "ghi-789", "name": "AI Mix", "description": "Generated by Claude"}
```

### Add an album to a playlist

```bash
python tidal_cli.py playlist add-album --playlist-id ghi-789 --album-id 1234
# Added 14 tracks from album "Discovery" to playlist "AI Mix".
```

### Add an individual track to a playlist

```bash
python tidal_cli.py playlist add-track --playlist-id ghi-789 --track-id 56789
# Added track "Around the World" to playlist "AI Mix".
```

## Complete typical workflow (LLM agent)

```bash
# 1. Find the artist
python tidal_cli.py --json search artist "Daft Punk"

# 2. Find the album
python tidal_cli.py --json search album "Discovery"

# 3. Create the playlist
python tidal_cli.py --json playlist create --name "Electronic Classics"

# 4. Add the album to the playlist
python tidal_cli.py playlist add-album --playlist-id <PLAYLIST_ID> --album-id <ALBUM_ID>
```

### Rename a playlist

```bash
python tidal_cli.py playlist rename --playlist-id ghi-789 --name "New Name"
# Renamed playlist "Electronic Classics" to "New Name".
```

### Remove a track from a playlist

```bash
python tidal_cli.py playlist remove-track --playlist-id ghi-789 --track-id 56789
# Removed track "Around the World" from playlist "New Name".
```

### Delete a playlist

```bash
python tidal_cli.py playlist delete --playlist-id ghi-789
# Deleted playlist "New Name".
```

### Manage the library (favorites)

```bash
# Add to library
python tidal_cli.py library add --artist-id 7804
# Added artist "Daft Punk" to library.

python tidal_cli.py library add --album-id 1234
# Added album "Discovery" to library.

python tidal_cli.py library add --track-id 56789
# Added track "Around the World" to library.

# Remove from library
python tidal_cli.py library remove --artist-id 7804
# Removed artist "Daft Punk" from library.
```

## Error Handling

- If the session has expired, commands automatically refresh the token.
- If the refresh token is expired, the error will instruct the administrator to re-run `auth`.
- All errors are written to stderr. Results are on stdout.
- Exit code: 0 = success, 1 = error.
